# 💳 Banking Statement System

This is a GUI-based Banking Statement System built using Python (Tkinter) and MySQL.

## 🚀 Features
- Add account details
- Store transactions (Deposit/Withdraw)
- Generate mini bank statement
- Update account information
- Display data in table format

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- MySQL (Database)

## ▶️ How to Run
1. Install Python
2. Install pymysql:
   pip install pymysql
3. Create MySQL database "bank"
4. Run the file:
   python banking_statement.py

## 📌 Note
Update database username & password in code before running.
